Todas as funções de cadeia de caracteres internas, exceto FORMAT, são determinísticas.All built-in string functions except FORMAT are deterministic. Isso significa que elas retornam o mesmo valor sempre que são chamadas com um conjunto específico de valores de entrada.This means they return the same value any time they are called with a specific set of input values. Para obter mais informações sobre determinismo de funções, consulte Funções determinísticas e não determinísticas.For more information about function determinism, see Deterministic and Nondeterministic Functions.

Quando as funções de cadeias de caracteres recebem argumentos que não são valores de cadeias de caracteres, o tipo de entrada é implicitamente convertido em um tipo de dados de texto.When string functions are passed arguments that are not string values, the input type is implicitly converted to a text data type. Para obter mais informações, consulte Conversão de tipo de dados (Mecanismo de Banco de Dados).For more information, see Data Type Conversion (Database Engine).
